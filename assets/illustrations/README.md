# Иллюстрации · Artwork Showcase

В этой папке лежат «иллюстрации» из блока ARTWORK основного [README.md](../../README.md).

## Что здесь

- `01.jpg`–`04.jpg` — рабочие копии, на которые ссылаются [README.md](../../README.md) и [README_EN.md](../../README_EN.md).
- Оригиналы с каомодзи в именах сохранены как backup: `Miu Miu.jpeg`, `☆⌒(＞。≪).jpeg`, `(´TωT\`).jpeg`, `_￠(･ω･｀).jpeg`. README на них не ссылается, потому что URL-encoding каомодзи нестабильно работает на мобильном GitHub.

## Как заменить на свои файлы

1. Положи свой ассет рядом, например `01.png`, `01.gif`, `01.webp` и т. д. Поддерживается всё, что рендерит GitHub Markdown (PNG, JPG, GIF, WebP, SVG).
2. Если расширение отличается от `.jpg` — поправь его в обоих README:

   ```html
   <img src="assets/illustrations/01.gif" />
   ```

3. Зафиксируй изменения и запушь.

## Рекомендации

- Цветовая гамма: чёрный фон, белый акцент, красный `#e60012` где уместно.
- Соотношения: hero — широкая (≈21:9 или 16:9), три квадрата — 1:1.
- Для GIF — 480p, до 5 секунд, до 1.5 МБ.
- Темы под текущий профиль: монохром / аниме-кадры / JDM-силуэты.
