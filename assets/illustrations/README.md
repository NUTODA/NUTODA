# Иллюстрации · Artwork Showcase

Эта папка соответствует блоку «Иллюстрации» из Steam-стиля профиля. По умолчанию здесь лежат 4 SVG-плейсхолдера — `01.svg`–`04.svg`.

## Как заменить на свои файлы

1. Положи свой ассет рядом, например `01.png`, `01.gif`, `02.jpg` и т. д. Поддерживаются все форматы, которые рендерит GitHub Markdown (PNG, JPG, GIF, WebP, SVG).
2. В корневых [README.md](../../README.md) и [README_EN.md](../../README_EN.md) поправь расширения путей на свои:

   ```html
   <img src="assets/illustrations/01.png" />
   ```

3. Зафиксируй изменения и запушь.

## Рекомендации по стилю (под Steam-вайб)

- Цветовая гамма: чёрный фон, тёплое золото (`#c8a96b`), приглушённые серые акценты.
- Соотношения: первые три — квадрат (1:1, ~640×640), четвёртая — широкая «герой-картинка» (2:1, ~1280×640), как `Miu Miu` в твоём Steam-профиле.
- Темы: монохромная аниме-эстетика (Madoka, K-On, любая «школьная форма»), JDM-силуэты (S15 Silvia, AE86), японская типографика (`快風刃零`, `ようこそ`).
- Размер файла: до 1.5 МБ на иллюстрацию, чтобы README быстро открывался; для GIF — 480p и до 5 секунд.

## Источник

Если хочется буквально перенести Steam-иллюстрации, скачай оригиналы со своей страницы:

- <https://steamcommunity.com/id/NUTODA/images/?appid=0>

Steam раздаёт их через CDN с временной подписью, поэтому ссылаться на них напрямую из README не получится — только локально через эту папку.
